#TeamCompound introduces compoundOS 3

January 14, 2022

compoundOS 3 is a new operating system that will be built from the ground up to be compatible with modern computing.

It is designed to provide a secure and stable platform for the future of computing. It will have a modular architecture that will enable developers to easily create custom modules, and it will be compatible with modern hardware.

It allows developers to write code for any platform and have it executed anywhere.


Learn more about the new operating system here: https://teamcompound.github.io/compoundos
